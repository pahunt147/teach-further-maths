What is 'Teach Further Maths'?

'Teach Further Maths' is a suite of high quality, fully animated colour A-Level Further Maths PowerPoint presentations, consisting of over 3000 slides.

Written for teachers by a very experienced classroom practitioner, examiner and published maths author (Paul Hunt), the presentations include demonstrations, proofs, worked examples, exercises, exam-style questions and actual exam questions (used with permission).

'Teach Further Maths' is perfect for the current specifications and includes complete coverage of the compulsory content from all of the main examination boards — Edexcel, AQA, OCR and MEI.

Each presentation is designed to be used directly in the classroom, saving teachers significant preparation time whilst delivering exceptional quality lessons.

Frequently Asked Questions

What software do I need to use 'Teach Further Maths'?

To use all features of the presentations, including any interactive features, you should use PowerPoint 2007 or later.

I would also suggest (although it isn't essential) that you install MathType, which is the equation editor used to write the presentations. Installing MathType will ensure that all equations are displayed correctly and will also give you the power to edit equations in the presentations should you wish to do so. After using the trial version for 30 days, it will revert to MathType Lite, which is still free and highly functional.

What if I have an older version of PowerPoint?

If you are using PowerPoint 2003 or earlier, some presentations may lose their fluency and some features (e.g. certain animations) may not function at all. Fluency can also be lost if you are running the presentations on an old or slow computer.

An excellent tool which improves compatibility is the Microsoft Office Compatibility Pack — free to install, it allows you to view and edit presentations created in newer versions of PowerPoint.

What if I don't have any version of PowerPoint?

You can still use the presentations! PowerPoint Viewer is a free tool that allows you to view fully-featured PowerPoint presentations without PowerPoint installed. It plays presentations flawlessly, although any interactive features or web links will be disabled.

What if I find any errors or omissions in the presentations?

Please contact us if you come across any errors or omissions. We will be very grateful for any comments or suggestions, and will be more than happy to make any necessary amendments and send you replacement presentations without fuss.
